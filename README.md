# 3D Models Collection

Welcome to the 3D Models Collection! This repository contains a variety of 3D models that can be used in various projects. Below is the documentation for one of our featured models, the **auraHeart**.

## auraHeart Model
 The **auraHeart** is a unique and visually striking 3D model designed to represent the essence of energy and passion. Its design incorporates elements that symbolize vitality and creativity.

### Features
- **Format**: OBJ, FBX
- **Polygons**: Approx. 15,000
- **Materials**: PBR materials included for realistic rendering.
- **Compatibility**: Usable in most 3D software like Blender, Maya, Unity, and Unreal Engine.

### Usage
To incorporate the auraHeart model into your project:
1. Download the model files from this repository.
2. Import the model into your chosen 3D software.
3. Adjust materials and lighting as needed.
